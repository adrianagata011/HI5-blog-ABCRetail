# Blog ABC Retail 
> Desarrollado por HiFive Developers 

## Instalación de módulos Node.js 

````` 
npm init -y
npm i express
npm i body-parser
npm i fs-extra
npm i bcrypt
npm i pug
npm i express-session
````` 

## Comandos para la ejecución del sistema 
> Utilizar el comando nº 1. 

1. Sin archivo bin 

````` 
node index
````` 

2. Con archivo bin 

````` 
node bin/www
`````
